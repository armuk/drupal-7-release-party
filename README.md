# Installation

* Download or clone the repo.
* Open index.html in a browser!

# Configuration

* Open index.html in a text editor and locate the config object.

      var config = {
        // What to search for.
        queries: ['drupal', 'd7rp'],
  
        // Flickr API key.
        flickr_api_key: 'YOUR API KEY',
  
        // UI update speed.
        ui_speed: 7000
      };

* Make any changes you would like and save it.  That's it!
