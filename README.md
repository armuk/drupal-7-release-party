# Installation

* Download or clone the repo.
* Open index.html in a browser!

# Configuration

* Open index.html and locate the config object.

    var config = {
      // Flickr tags to pull images from.
      queries: ['drupal', 'd7rp'],
  
      // Flickr API key.
      flickr_api_key: 'YOUR API KEY',
  
      // UI update speed.
      ui_speed: 7000
    };

* Save your changes!  That's it!
